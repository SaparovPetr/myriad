# run into - встретить случайно

Marked list of seven simple and popular collocations with "run into" (встретить случайно):

1. run into a friend  
2. run into an old classmate  
3. run into someone at the store  
4. run into trouble  
5. run into difficulties  
6. run into a celebrity  
7. run into an ex  

Examples in different tenses:

Present Perfect: I have run into my neighbor twice this week.  
Past Simple: Yesterday, I ran into an old friend from college.  
Past Continuous: I was running into the building when I saw her.  
Past Perfect: By the time I arrived, he had already run into some problems.
