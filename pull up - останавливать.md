# pull up - останавливать

1. pull up a chair
2. pull up to the curb
3. pull up a file
4. pull up the handbrake
5. pull up at the traffic lights
6. pull up a website
7. pull up weeds

- Present Perfect: The taxi has just pulled up outside the building.
- Past Simple: He pulled up the document on his computer to show me the data.
- Past Continuous: While I was pulling up the weeds, it started to rain.
- Past Perfect: She had already pulled up to the station before she realized her mistake.
